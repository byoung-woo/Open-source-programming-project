# 🎓 Open Source Programming Project - Final Exam

이 저장소는 오픈소스 프로그래밍 기말고사 과제 풀이를 담고 있습니다.  
총 5개의 알고리즘 문제에 대한 해결 코드가 Python으로 작성되어 있습니다.

## 🛠️ 기술 스택
- **Language**: Python 3
- **Libraries**: `math` (Pro4.py)

## 📂 파일 구성 및 문제 설명

### 1. [Pro1.py] 부분 문자열 확인하기
- **문제**: 문자열 `my_string`과 `target`이 주어졌을 때, `target`이 `my_string`의 부분 문자열인지 확인합니다.
- **기능**:
  - `target`이 포함되어 있으면 `1`, 아니면 `0`을 반환합니다.
  - 입력값의 길이 및 소문자 여부에 대한 예외 처리(`MyError`)가 포함되어 있습니다.

### 2. [Pro2.py] 모스 부호 해독기
- **문제**: 모스 부호로 이루어진 문자열 `letter`를 매개변수로 받아 영어 소문자 문장으로 해독합니다.
- **기능**:
  - 공백으로 구분된 모스 부호를 알파벳으로 변환합니다.
  - 잘못된 입력(길이 초과, 연속된 공백 등)에 대한 예외 처리를 수행합니다.

### 3. [Pro3.py] PROGRAMMER-857 행성의 나이 변환
- **문제**: 숫자로 된 나이를 특정 행성의 알파벳 표기법(a=0, b=1, ..., j=9)으로 변환합니다.
- **기능**:
  - 입력받은 자연수 `age`를 대응하는 알파벳 문자열로 변환하여 반환합니다. (예: 23 -> "cd")
  - `isinstance` 등을 활용한 입력값 유효성 검사를 수행합니다.

### 4. [Pro4.py] 두 원 사이의 정수 좌표 개수 구하기
- **문제**: 중심이 원점인 서로 다른 크기의 두 원(반지름 `r1`, `r2`) 사이(경계 포함)에 존재하는 정수 좌표(x, y)의 개수를 구합니다.
- **기능**:
  - `math` 모듈을 사용하여 1사분면의 정수 좌표 개수를 계산한 뒤, 대칭성을 이용해 전체 개수를 도출합니다.
  - 두 원 사이의 공간 면적 내에 존재하는 점들을 효율적으로 계산합니다.

### 5. [Pro5.py] 가장 큰 수 만들기
- **문제**: 0 또는 양의 정수가 담긴 배열 `numbers`가 주어졌을 때, 순서를 재배치하여 만들 수 있는 가장 큰 수를 문자열로 반환합니다.
- **기능**:
  - 숫자들을 문자열로 변환한 뒤, 조합 시 크기 비교를 위해 4자리로 맞춰 정렬하는 로직을 사용합니다.
  - `sorted` 함수의 `key` 파라미터를 커스텀하여 정렬 기준을 정의했습니다.

## 🚀 실행 방법

각 파일은 독립적으로 실행 가능하며, 파일 내부의 `while True` 루프나 테스트 코드를 통해 입력을 받고 결과를 출력합니다.

```bash
# 예시: 1번 문제 실행
python Pro1.py
```

예외 상황 발생 시 커스텀 예외 클래스 `MyError`를 통해 적절한 에러 메시지를 출력합니다.
