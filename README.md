# 🎓 Open Source Programming Final Project

이 저장소는 2023학년도 오픈소스 프로그래밍 기말고사 과제 풀이 및 보고서를 포함하고 있습니다.

## 📂 프로젝트 구성 및 문제 설명

### 1. [Pro1.py] 문자열 포함 여부 확인 (String Subset)
- **문제**: `target` 문자열이 `my_string`의 부분 문자열인지 판단하여 1 또는 0을 반환합니다.
- **접근 방식**:
  - 파이썬의 `in` 연산자를 사용하여 포함 여부를 직관적으로 판별했습니다.
  - `while True` 루프와 `input()` 함수를 사용하여 사용자가 직접 값을 입력하고 결과를 확인할 수 있도록 심화 구현했습니다.
- **주요 기능**:
  - **예외 처리 (`MyError`)**: 입력 길이(1~100) 초과 및 대문자 입력 시 사용자 정의 에러 메시지를 출력하고 재입력을 유도합니다.

### 2. [Pro2.py] 모스 부호 해독기 (Morse Code Decoder)
- **문제**: 공백으로 구분된 모스 부호 문자열을 입력받아 영어 소문자 문장으로 변환합니다.
- **접근 방식**:
  - `split(' ')` 함수를 사용하여 모스 부호를 개별 문자로 분리했습니다.
  - 딕셔너리(Dictionary) 자료구조를 활용해 모스 부호와 알파벳을 매핑하여 변환 속도를 높였습니다.
- **주요 기능**:
  - 연속된 공백이나 길이 제한을 위반할 경우 예외를 발생시켜 입력 데이터의 무결성을 검증합니다.

### 3. [Pro3.py] 행성 나이 변환 (Planet Age Converter)
- **문제**: 숫자로 된 나이를 특정 행성의 알파벳 표기법(0='a', 1='b', ..., 9='j')으로 변환합니다.
- **접근 방식**:
  - 나이(`age`)를 10으로 나눈 나머지(%)를 인덱스로 활용하여 `alpha` 리스트에서 문자를 매핑했습니다.
  - 몫(`//`) 연산을 반복하여 모든 자릿수를 변환했습니다.
- **주요 기능**:
  - **입력 검증**: `isinstance()`와 `isdigit()` 함수를 사용하여 정수가 아닌 값(소수, 문자 등) 입력 시 에러를 처리합니다.

### 4. [Pro4.py] 두 원 사이의 정수 좌표 개수 (Points in Circle)
- **문제**: 중심이 같은 두 원(반지름 r1, r2) 사이의 공간에 존재하는 정수 좌표(x, y)의 개수를 구합니다.
- **접근 방식**:
  - 피타고라스 정리를 이용해 1사분면의 y좌표 높이를 `math.sqrt`로 계산했습니다.
  - `math.ceil`(올림)을 사용하여 y축 아래의 정수 개수를 효율적으로 카운트했습니다.
  - **정밀도 문제 해결**: `math.sqrt` 결과가 실수형이므로, 원 위의 점을 정확히 포함하기 위해 `int(val) == val` 비교 구문을 사용하여 경계값 처리를 보완했습니다.
- **특이 사항**: 1사분면의 개수를 구한 뒤 대칭성을 이용해 전체 개수(`* 4`)를 계산했습니다.

### 5. [Pro5.py] 가장 큰 수 만들기 (Make Largest Number)
- **문제**: 주어진 정수 배열을 조합하여 만들 수 있는 가장 큰 수를 문자열로 반환합니다.
- **접근 방식**:
  - 단순히 숫자의 크기로 정렬하는 것이 아니라, 문자로 변환 후 앞자리가 큰 순서대로 이어 붙여야 합니다.
  - `sorted()` 함수의 `key` 파라미터에 커스텀 함수(`compare`)를 적용했습니다. 숫자를 문자열로 변환하고 4번 반복(`str(x)*4`)하여 4자리까지 비교함으로써 3과 30 중 3이 먼저 오도록 정렬 기준을 맞췄습니다.
- **주요 기능**:
  - 입력값이 모두 0인 경우(예: `[0, 0]`) "00"이 아닌 "0"을 반환하도록 예외 처리를 추가했습니다.
